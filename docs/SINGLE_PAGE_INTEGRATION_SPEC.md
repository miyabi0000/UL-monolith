# ギアCardギア管理システム 要件定義書

## 概要
GEAR ANALYSISの右側パネルを多機能化し、チャートのリアルタイムディスクリプションとして機能させる。
下部のGEAR LISTは維持。既存のレイアウトや表記を尊重
コンポーネントは既存のものを流用。サイズは変更して、既存のレイアウトを崩さない。
---

## レイアウト
既存のレイアウトを崩さないようにする。

```
┌────────────────────────────────────────────────────────┐
│ GEAR ANALYSIS    [+ Add] [Weight/Cost] [モード切替]   │
├──────────────────────────┬─────────────────────────────┤
│                          │ 右側パネル（3モード）       │
│   [Pie Chart]            │                             │
│                          │ ①アイテム詳細               │
│   - カテゴリクリック     │ ②カテゴリサマリー           │
│   - アイテムクリック     │ ④全体サマリー               │
│                          │                             │
└──────────────────────────┴─────────────────────────────┘
│ GEAR LIST（維持）                                      │
└────────────────────────────────────────────────────────┘
```

---

## 右側パネルの3モード（チャートのリアルタイムディスクリプション）

### ①アイテム詳細（Item Detail）
**トリガー**: Pie Chartのアイテムをクリック

**表示**:
- 画像、名前、ブランド、カテゴリ
- 重量、価格、Own/Need/Shortage、優先度、シーズン
- ホバーで編集/削除/URLボタン

---

### ②カテゴリサマリー（Category Summary）
**トリガー**: Pie Chartのカテゴリラベルをクリック

**表示**:
すべて英語
```
[カテゴリ名]
━━━━━━━━━━━━━━
 統計
- アイテム数: 3個
- 合計重量: 450g
- 合計価格: ¥12,000

━━━━━━━━━━━━━━
アイテムリスト（縦スクロール）
[アイテムカード1] → クリックで①へ
[アイテムカード2]
[アイテムカード3]
```

---



### ④全体サマリー（Overview）
**トリガー**: 初期表示、または選択解除

**表示**:
```

総重量: 1,870g
総価格: ¥176,702
アイテム数: 13個
不足数: 13個

━━━━━━━━━━━━━━
統計
平均重量: 144g
最重量: テント (850g)
最高価格: テント (¥45,000)

━━━━━━━━━━━━━━
アクション推奨
[不足アイテムを表示] → ③へ
[優先度「高」を表示] → ③へ
```
