# ギアリスト一括編集機能 - 要件定義書

## 📋 概要

### 目的
テーブルビューとカードビューの両方で、複数のギアアイテムを選択して一括操作（削除・カテゴリ変更など）ができる機能を提供する。

### 背景
- チェックボックスモードの実装は既に完了している
- UIの起点となるメニューボタンがない
- ユーザーがチェックボックスモードを発見・利用しづらい

---

## 🎯 機能要件

### 1. スリードットメニュー（操作メニュー）
ギアリストの右上にメニューボタンを配置

#### 配置場所
```
┌─────────────────────────────────────────────────┐
│  Gear Items (10)  [ソート] [フィルター]   [⋮]  │ ← ここ
└─────────────────────────────────────────────────┘
```

#### メニュー項目
- **✓ 一括編集モード** - チェックボックス表示/非表示の切替
- **↻ 更新** - リスト再読み込み
- **⚙️ 設定** - （将来的な拡張用）

### 2. 一括編集モードフロー

#### Step 1: メニューから有効化
1. スリードットメニュー（⋮）をクリック
2. 「一括編集モード」を選択
3. チェックボックスが各アイテムに表示される

#### Step 2: アイテム選択
1. 個別にチェックボックスをクリック
2. または「すべて選択」で全選択
3. ヘッダーに選択数を表示: 「3個選択中」

#### Step 3: 一括操作実行
選択後に表示される操作ボタン：
- **削除** - 選択したアイテムを一括削除
- **カテゴリ変更** - 選択したアイテムのカテゴリを一括変更（将来的）
- **キャンセル** - 選択をクリア

### 3. UI表示状態

#### 通常モード（チェックボックスなし）
```
┌─────────────────────────────────────────────────┐
│  Gear Items (10)                          [⋮]   │
├─────────────────────────────────────────────────┤
│  [Name]  [Category]  [Weight]  [Price] [Edit]  │
│  Tent    Shelter     1.2kg     ¥45,000  [✎]    │
│  Bag     Backpack    850g      ¥32,000  [✎]    │
└─────────────────────────────────────────────────┘
```

#### 一括編集モード（チェックボックスあり）
```
┌─────────────────────────────────────────────────┐
│  ☑ すべて選択 | 3個選択中      [削除] [キャンセル] │
├─────────────────────────────────────────────────┤
│  [☑] [Name]  [Category]  [Weight]  [Price]     │
│  [☑] Tent    Shelter     1.2kg     ¥45,000     │
│  [☑] Bag     Backpack    850g      ¥32,000     │
│  [ ] Jacket  Clothing    320g      ¥28,000     │
└─────────────────────────────────────────────────┘
```

### 4. カードビューでの表示

#### 通常モード
```
┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
│[IMG] │  │[IMG] │  │[IMG] │  │[IMG] │
│Tent  │  │Bag   │  │Jacket│  │Stove │
└──────┘  └──────┘  └──────┘  └──────┘
```

#### 一括編集モード
```
┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
│☑[IMG]│  │☑[IMG]│  │ [IMG]│  │☑[IMG]│
│Tent  │  │Bag   │  │Jacket│  │Stove │
└──────┘  └──────┘  └──────┘  └──────┘
     ↑ チェックボックスが左上に表示
```

## 🔧 技術要件

### 新規コンポーネント

#### 1. `BulkActionMenu.tsx` - スリードットメニュー
```typescript
interface BulkActionMenuProps {
  showCheckboxes: boolean;
  onToggleCheckboxes: () => void;
  onRefresh: () => void;
}
```

**機能**:
- スリードットアイコンボタン
- ドロップダウンメニュー
- メニュー項目のクリックハンドラー

#### 2. `BulkActionBar.tsx` - 一括操作バー
```typescript
interface BulkActionBarProps {
  selectedCount: number;
  totalCount: number;
  onSelectAll: () => void;
  onClearSelection: () => void;
  onBulkDelete: () => void;
  // onBulkCategoryChange?: () => void; // 将来的
}
```

**機能**:
- 選択数表示
- 「すべて選択」チェックボックス
- 一括削除ボタン
- キャンセルボタン

### 既存コンポーネントの更新

#### 1. `GearTable.tsx`
- ヘッダーに`BulkActionMenu`を追加
- チェックボックスモード時に`BulkActionBar`を表示
- 既存の`showCheckboxes`ロジックと統合

#### 2. `GearView.tsx`
- ヘッダーに`BulkActionMenu`を追加
- チェックボックスモード時に`BulkActionBar`を表示
- 既存の`showCheckboxes`ロジックと統合

#### 3. `App.tsx`
- `showCheckboxes`のtoggle関数を提供
- リスト更新関数を提供

### 状態管理

既存の状態を活用：

## 🎨 UI/UXデザイン

### スリードットメニュー
- **アイコン**: `⋮` (縦3点)
- **位置**: 右上、既存ボタン群の右
- **スタイル**: 
  - ホバー時: 背景色が薄いグレーに
  - クリック時: ドロップダウン展開
  - ドロップダウン: 白背景、影付き

### 一括操作バー
- **表示位置**: テーブル/グリッドの直上
- **背景色**: 薄い青（プライマリーカラー）
- **高さ**: 48px
- **レイアウト**: 左側に選択情報、右側にアクションボタン

### ボタンスタイル
- **削除ボタン**: 赤色（COLORS.danger）
- **キャンセルボタン**: グレー
- **すべて選択**: チェックボックス + ラベル

## 🔄 バックエンドAPI（既存）

一括削除は既存APIを複数回呼び出す方式：

```typescript
// フロントエンド側
const handleBulkDelete = async (selectedIds: string[]) => {
  await Promise.all(selectedIds.map(id => 
    fetch(`/api/gear/${id}`, { method: 'DELETE' })
  ));
};
```

**既存エンドポイント**:
- `DELETE /api/gear/:id` - 単一削除（既存）

**将来的な拡張**:
- `POST /api/gear/bulk-delete` - 一括削除専用
- `PATCH /api/gear/bulk-update` - 一括更新
